# Aditya Dresses - Premium E-Commerce Platform 🛍️

A robust, full-stack E-Commerce web application built to deliver an ultra-premium shopping experience. It features dynamic routing, state-driven interfaces, and a bespoke Express.js backend for tracking live analytics.

## 🚀 Key Features

* **Multi-Step Secure Checkout**: A Flipkart-inspired interactive checkout pipeline that securely collects delivery addresses and simulates UPI, COD, and Card payment processing.
* **Live Order Tracking**: A dedicated tracking utility mapping the lifecycle (Processing → Shipped → Delivered) of user shipments using a custom order ID structure.
* **Intelligent User Dashboards**: Secure Authentication (Signup/Login) system generating private, interactive dashboards detailing past purchase history.
* **Dynamic Cart Management**: Real-time badge polling on the navigation bar, secure item validation, and dynamic total calculation algorithms.
* **Live Search Filtering**: An instant, keystroke-responsive search engine filtering across a diverse catalog of 60 premium items.
* **"Boutique" Design System**: Completely custom UI styled with global CSS variables, dark-mode luxury aesthetics, dynamic glassmorphism, and Framer Motion animations.

## 🛠️ Technology Stack

* **Frontend**: React.js, Vite, React-Router-Dom, Framer Motion, Lucide-React
* **Backend**: Node.js, Express.js, CORS
* **Database Management**: Dynamic REST API reading/writing synchronously to a structured JSON framework.
